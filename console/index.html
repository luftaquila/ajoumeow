  • 현재 학기 값은 학기가 바뀌면 항상 업데이트 해 주세요.
         명단 작성 등 전산 여러 곳에서 중요하게 사용되는 값입니다.
  • 공지사항은 값을 변경할 때마다 급식표 페이지 방문 시 자동으로 팝업을 표시합니다. 자동 팝업은 값 변경 후 최초 1회만 표시됩니다. 이후에는 사용자가 공지사항 메뉴를 눌러야 팝업이 보여집니다.

    • 회원 등록과 신입 모집은 오늘 날짜가 설정 기간에 포함되거나
    추가 허용 여부가 TRUE일 때, 둘 중 하나만 만족해도 활성화됩니다.

    추가 허용 항목은 평소에는 FALSE로 맞춰 놓고
    기간외 등록을 희망하는 사람이 있을 때 잠깐 TRUE로 바꿨다가
    해당 사람이 등록을 마치면 다시 FALSE로 바꾸면 됩니다.

    회원 관리

  • 회원 목록을 관리합니다. 학번 외 모든 항목은 수정할 수 있습니다.
  • 수정할 값을 클릭하면 입력창이 활성화됩니다.
          수정할 값을 입력한 후 엔터를 누르면 수정됩니다.
  • 직책이 '회원'이 아닌 사람에게는 모두 관리자 권한이 부여됩니다.
          관리자는 관리자 페이지 접속 권한, 급식표 모든 셀의 편집 권한을 갖습니다. 또한 본인 외 다른 신분으로 급식을 신청할 수 있습니다.
